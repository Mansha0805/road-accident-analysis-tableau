# Road Accident Analysis Dashboard (Tableau)

A visual analytics dashboard built in **Tableau** to explore road accident data, uncover patterns, and generate insights for road safety interventions.


## Motivation

Understanding road accident patterns is critical for improving traffic safety and reducing fatalities. Visual dashboards help transform raw accident records into actionable insights, making it easier for stakeholders to spot trends, hotspots, and risk factors.

---

## Features

- Interactive filters by **location**, **accident severity**, **vehicle type**, **weather conditions**, etc.  
- Map visualizations showing spatial distribution and hotspot regions  
- Trend charts for temporal patterns (e.g. accidents over years)  
- Bar / pie charts for categorical breakdowns (e.g. types of vehicles involved, cause categories)
